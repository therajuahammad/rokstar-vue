Node.js and Yarn

Your computer must have installed nodejs and yarn to run this template. You can download Node.js from https://nodejs.org and yarn from https://yarnpkg.com/lang/en/docs/install/. NPM comes bundled with Node.js







Install Npm Packages

You must have to install packages. You can do this by running yarn install from the root of your project to install all the necessary dependencies.




Development server

Run yarn dev from the root of your project to start a dev server. Navigate to http://localhost:3000/. The app will automatically reload if you change any of the source files.




Server Deployment

Run yarn build. To deploy a SSR application we use target: 'server', where server is the default value. Nuxt will create a .nuxt directory with everything inside ready to be deployed on your server hosting. Once your application is built you can use the start command to see a production version of your application.




Static Deployment (Pre-rendered)

Run yarn generate. Nuxt gives you the ability to host your web application on any static hosting. To deploy a static generated site make sure you have target: 'static' in your nuxt.config.js (for Nuxt >= 2.13). Nuxt will create a dist/ directory with everything inside ready to be deployed on a static hosting service.


https://v2.nuxt.com/docs/get-started/commands/

Page Structure

'./pages/index.js'

The pages directory contains your Application Views and Routes. Nuxt reads all the .vue files inside this directory and automatically creates the router configuration for you. https://v2.nuxt.com/docs/directory-structure/pages/



Components Structure

'/components'

The components directory contains your Vue.js Components. Components are what makes up the different parts of your page and can be reused and imported into your pages, layouts and even other components. https://v2.nuxt.com/docs/directory-structure/components/

Contact form Email Configuration

For Sending email dynamically we used EmailJS library.

For the configuration we need Service ID, Template ID, and Public Key. We will get these ids on EmailJS dashboard after the signup.







Need some variable for create a template on EmailJS
  • For get Name: "name"
  • For get Email: "email"
  • For get Subject: "subject"
  • For get Budget: "budget"
  • For get Message: "message"
Please follow for Details: https://www.emailjs.com/docs/sdk/send-form/
